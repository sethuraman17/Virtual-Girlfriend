⚙️ Step 1: Uninstall CPU-only PyTorch

In your virtual environment:

pip uninstall torch torchvision torchaudio -y

⚙️ Step 2: Install the correct GPU-enabled PyTorch

Since your driver supports CUDA 12.9, you should install PyTorch built for CUDA 12.1, which works perfectly fine (PyTorch doesn’t yet release CUDA 12.9 wheels).

Run:

pip install torch torchvision torchaudio --index-url https://download.pytorch.org/whl/cu121


That will take a few minutes — it’s around 3–4 GB of packages.

⚙️ Step 3: Verify the installation

After installation completes, run:

python -c "import torch; print(torch.__version__); print(torch.cuda.is_available()); print(torch.cuda.get_device_name(0) if torch.cuda.is_available() else 'No GPU')"


✅ Expected output:

2.4.1
True
NVIDIA GeForce GTX 1650


If you see that, PyTorch is now properly connected to your GPU.
